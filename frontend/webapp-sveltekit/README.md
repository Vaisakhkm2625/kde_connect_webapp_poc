# KDE Connect Webapp - SvelteKit Frontend

A modern, desktop-like web interface for KDE Connect, built with SvelteKit and Tailwind CSS.

## Features
- **Window Management**: Draggable windows, taskbar, system tray.
- **Device Management**: Pair, ping, ring, and unpair devices.
- **Notifications**: View and send notifications.
- **Commands**: Execute remote commands and manage local scripts.
- **Tools**: Clipboard sync, Media player controls, Camera, File Browser configuration.

## Development

### Prerequisites
- Node.js 22+
- pnpm

### Setup
```bash
pnpm install
```

### Run
```bash
pnpm run dev
```

### Build
```bash
pnpm run build
```
